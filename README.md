# ☕ Brew & Bloom Café – Hyderabad

A modern, responsive landing page for an artisanal café located in Banjara Hills, Hyderabad. This project focuses on a "Caffeinated" aesthetic using warm tones, elegant typography, and smooth CSS animations.

## ✨ Key Features
* **Fully Responsive:** Optimized for Desktop, Tablet, and Mobile views.
* **Modern UI/UX:** Features Glassmorphism navigation and custom SVG illustrations.
* **Interactive Menu:** Grid-based layout with hover effects and "Order" triggers.
* **Smooth Animations:** Custom `@keyframes` for fade-in effects on scroll.
* **Direct Integration:** functional WhatsApp chat button and Google Maps embed.
* **Performance:** Lightweight single-file structure (HTML/CSS/JS).

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for SEO.
* **CSS3:** Custom properties (Variables), Flexbox, and CSS Grid.
* **JavaScript:** Mobile menu logic, scroll-based navbar shadows, and form validation.
* **Typography:** Playfair Display (Serif) & DM Sans (Sans-serif) via Google Fonts.
